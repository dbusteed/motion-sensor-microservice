Motion Sensor Microservice

/ This page
/addEvent accepts POST requests with motion event data
/events accepts GET requests and returns event data
/events?QUERY_PARAMS 
id
 return events that match id
ex: ?id=3 
deviceID
 return events that match deviceID
ex: ?deviceID=42 
deviceName
 return events where param in deviceName
ex: ?deviceName=door 
lte_timestamp
 return events where events less than or equal to timestamp
ex: ?lte_timestamp=1590335155 
gte_timestamp
 return events where events greater than or equal to timestamp
ex: ?gte_timestamp=1590335155

can combine multiple query parameters
